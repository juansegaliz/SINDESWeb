# SINDES - FESC

Sitio web del Semillero de Investigación en Desarrollo de Software e Inteligencia Artificial de la Fundación de Estudios Superiores Comfanorte.

## Tecnologías

- Angular 20 (Standalone Components)
- Tailwind CSS
- TypeScript
- Reactive Forms
- Signals API

## Instalación

```bash
npm install
```

## Desarrollo

```bash
npm start
```

Navega a `http://localhost:4200/`

## Build

```bash
npm run build
```

## Características

- ✅ 10 páginas completas con routing
- ✅ Componentes reutilizables
- ✅ Diseño responsive
- ✅ Accesibilidad (ARIA)
- ✅ SEO básico
- ✅ Lazy loading
- ✅ Formulario de contacto validado
- ✅ Preparado para SSR

## Estructura

```
src/
├── app/
│   ├── core/          # Servicios y modelos
│   ├── shared/        # Componentes UI reutilizables
│   ├── layout/        # Header, Footer, Shell
│   └── pages/         # 10 páginas del sitio
├── styles.css         # Estilos globales + Tailwind
└── main.ts
```
